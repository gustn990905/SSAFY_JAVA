# SSAFY JAVA

SSAFY(Java Track) 학습 과정에서 수행한 실습, 워크샵, 과제 및 예제 코드를 정리한 저장소입니다.

## 학습 목표

* Java 기본 문법 이해
* 객체지향 프로그래밍(OOP) 개념 학습
* 상속, 다형성, 인터페이스 활용
* 컬렉션 프레임워크 활용
* 예외 처리 및 파일 입출력 학습
* 알고리즘 문제 해결 능력 향상
